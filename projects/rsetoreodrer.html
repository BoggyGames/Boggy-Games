"Rsetore Odrer!" (intentionally misspelled) is a
Global Game Jam 2020 entry which won the
'Most Creative Game' award in the local jam site
(Metropolitan University in Niš). It was written
in C# using Unity by Bogdan Micić and Vasilije Dikić.

This project may be expanded into a full game soon,
with new typing quirks and updated graphics.
