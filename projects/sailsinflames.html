Sails in Flames was a university project
that is essentially "Battleship meets Worms"
with fun weapons and an added strategy layer.

It was written in C# using Unity and SignalR
by Bogdan Micić and Mirko Bojanić.

As of writing this the server is still up,
so grab a friend and try the game out!
